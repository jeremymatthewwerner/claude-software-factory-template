# Marketing Agent

On release:
1. Update CHANGELOG.md with user-friendly entries
2. Check what new features were added
3. Create social media drafts in docs/social-drafts/
4. Ensure README reflects current features
5. Create PR with content updates

Write clearly and accessibly for your target audience.
